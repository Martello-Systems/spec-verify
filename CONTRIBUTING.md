# Contributing to spec-verify

Thanks for your interest in improving `spec-verify` — an open-source tool by [Martello Systems](https://martellosystems.com).

## Ways to contribute
- 🐛 Report bugs via Issues
- 💡 Propose features or improvements
- 📖 Improve the docs
- 🔧 Submit pull requests

## Pull requests
1. Fork the repo and branch from `main`.
2. Make a clear, focused change.
3. Add or update tests where it makes sense.
4. Open a PR describing **what** changed and **why**.

By contributing, you agree that your contributions are licensed under the project's [Apache License 2.0](LICENSE).

## Conduct
Be respectful and constructive. We read and review every PR.
